## Project Scope

The project scope is limited to technical hiring roles such as:

- Software Engineer
- Web Developer
- Data Analyst
- Data Scientist
- Machine Learning Engineer

This allows better skill extraction, more accurate question generation, and a practical implementation within the internship timeline.
